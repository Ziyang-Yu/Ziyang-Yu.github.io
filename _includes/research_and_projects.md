
## Projects

**ResearchAtlas**: A coverage-oriented, graph-based academic paper discovery non-commercial website I designed and built to improve literature review beyond traditional Top-K ranking search. You are welcome to visit the [project website](https://www.researchatlas.dpdns.org/) for more details.

**LLMCostCut**: A framework for selective LLM invocation and online distillation to reduce the cost of using large language models while maintaining accuracy. It routes queries through a small student model and falls back to the teacher LLM only when the student is uncertain, achieving around 10× cost reduction and nearly 1000× speedup with accuracy up to 95%. You can find the code and documentation on [GitHub](https://github.com/zhaoliangvaio/llmcostcut).
